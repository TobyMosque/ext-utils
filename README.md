Quasar App Extension - TobyMosque Utilities
===

# Install
```bash
quasar ext add @toby.mosque/utils
```
Quasar CLI will retrieve it from NPM and install the extension.

# Uninstall
```bash
quasar ext remove @toby.mosque/utils
```
